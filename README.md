# Projeto Ciência de Dados

Este é um projeto de ciência de dados em Python.

## Estrutura do Projeto

- `data/`: Pasta para armazenar os dados do projeto (datasets, arquivos CSV, etc.)
  - `processed/`: Dados processados
- `notebooks/`: Notebooks Jupyter para análise exploratória e modelagem
  - `sample_notebook.ipynb`: Notebook de exemplo para automação de criação de arquivos
  - `analysis.ipynb`: Notebook para análises
- `src/`: Scripts Python para processamento de dados, modelos, etc.
  - `main.py`: Script principal
  - `utils.py`: Funções utilitárias
- `config/`: Arquivos de configuração
  - `config.yaml`: Configurações do projeto
- `models/`: Modelos treinados e artefatos de ML

## Configuração do Ambiente

1. Certifique-se de ter Python instalado (versão 3.8 ou superior recomendada).
2. Crie um ambiente virtual: `python -m venv .venv`
3. Ative o ambiente: `.venv\Scripts\activate` (Windows) ou `source .venv/bin/activate` (Linux/Mac)
4. Instale as dependências: `pip install -r requirements.txt`

## Uso

- Coloque seus dados na pasta `data/`
- Use os notebooks em `notebooks/` para análise
- Scripts utilitários em `src/`

## Contribuição

Adicione suas análises e modelos conforme necessário.
